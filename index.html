Resume Extraction and Summarization with NER

Introduction

The application we want to focus on is the automatic resume summarizer. This is an interesting problem because most resumes do not even get past the applicant tracking system. It causes unnecessary stress for both the recrutiers and applicants. We can use a NER model in this case to automatically mark the information on the resume corresponding to certain categories.

What is NER?

Named Entity Recognition (NER) has always been important in downstream tasks such as building a knowledge graph, user internet modeling, question answering, dialogue systems, etc. as it allows us to extract real-world entities from text and label them using predefined types.
